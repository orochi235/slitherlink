Slitherlink Solving Lemmas

A structured reference for logical deductions in Slitherlink (スリザーリンク) puzzles

Rules of Slitherlink

  • The grid is made of dots (vertices) connected by potential edges.
  • Draw a single continuous loop along the grid edges.
  • The loop cannot cross or branch — every vertex on the loop has exactly 2 edges.
  • Cells may contain a clue number (0–3) indicating how many of their 4 edges are part of the loop.
  • Cells without a number have no constraint on how many edges are used.

See axioms below for basic operations.

LEMMA 1

Corner three

A 3 in a corner cell forces all edges except the one at the outer corner vertex. That vertex has only two edges and both border the 3-cell, but only one can be used — so exactly one of them is in the loop, and the other two edges of the 3 are also forced.

LEMMA 2

Row of threes

In a row of adjacent 3 cells, every shared edge between them is in the loop, and the two outer edges parallel to the shared edges (top and bottom of the first and last cell) are also forced.

⟹ This extends to any number of consecutive 3s in a line.
LEMMA 3

Diagonal threes

Two 3 cells sharing a vertex (diagonally adjacent) force both edges of the shared vertex that don't border either 3-cell, since the shared vertex must have exactly 2 loop edges and both 3-cells demand edges there.

LEMMA 4

One loop to rule them all

An edge that would close the loop before all clues are satisfied must be eliminated. The loop must visit enough vertices to satisfy every clue.

Axioms — basic operations
AXIOM 1

Zero elimination

A cell with clue 0 has no loop edges — all four of its edges are immediately eliminated.

AXIOM 2

Clue satisfaction

If a cell already has N edges in the loop matching its clue N, all remaining edges of that cell are eliminated.

AXIOM 3

Remaining forced

If a cell with clue N has exactly N undetermined edges remaining (and the rest are eliminated), all remaining edges must be in the loop.

AXIOM 4

Vertex parity

Every vertex has 0 or 2 loop edges. If a vertex already has 2 loop edges, all other edges at that vertex are eliminated. If it has 1 loop edge and only 1 undetermined edge remains, that edge is forced.

AXIOM 5

No junctions

If two loop edges pass straight through a vertex, the perpendicular edges at that vertex must be eliminated — the loop cannot branch.

AXIOM 6

Dead end

If a vertex has only one undetermined edge and no loop edges, that last edge must be eliminated (a single edge cannot form a loop).
