<sl-diagram
  size="3x3"
  clues="1,1:3 2,1:2 3,1:1 1,2:2 2,2:0 3,2:3 1,3:1 2,3:2 3,3:2"
  continue="none"
></sl-diagram>

Active lines (loop edges)

A complete loop around a 2×2 grid.

<sl-diagram
  size="2x2"
  clues="1,1:2 2,1:2 1,2:2 2,2:2"
  lines="0,0->RRDDLLU"
  continue="none"
></sl-diagram>

Eliminated edges — gaps (default)

Eliminated edges are removed, leaving gaps in the grid.

<sl-diagram
  size="2x2"
  clues="1,1:1 2,2:1"
  erased="0,0->RD; 2,0->D; 0,2->R; 1,1->RD"
  continue="none"
></sl-diagram>

Explicit X markers

Red ✕ markers via x-marks attribute. Edges are removed from the grid and marked.

<sl-diagram
  size="2x2"
  clues="1,1:1 2,2:1"
  x-marks="0,0->RD; 2,0->D; 0,2->R; 1,1->RD"
  continue="none"
></sl-diagram>

Emphasized (forced) edges

Goldenrod edges with matching endpoint dots, drawn on top of everything.

<sl-diagram
  size="2x2"
  clues="1,1:3 2,2:3"
  emphasis="0,0->R; 0,0->D; 1,2->R; 2,1->D"
  continue="none"
></sl-diagram>

Lines + emphasis combined

Active loop in black, forced deductions in goldenrod.

<sl-diagram
  size="3x3"
  clues="1,1:3 2,1:2 3,1:3"
  lines="0,0->RRR; 0,0->D; 3,0->D"
  emphasis="0,1->R; 2,1->R"
  continue="bottom"
></sl-diagram>

Captions

Per-diagram captions via the caption attribute.

<sl-diagram
  size="2x2" clues="1,1:3"
  continue="none"
  caption="Before"
></sl-diagram>

<sl-diagram
  size="2x2" clues="1,1:3"
  lines="0,0->RD"
  emphasis="1,0->D; 0,1->R"
  continue="none"
  caption="After"
></sl-diagram>

Sequence (sl-sequence)

Multiple diagrams with arrow separators and synced extents.

<sl-sequence>
  <sl-diagram
    size="2x2" clues="1,1:3 2,2:3"
    continue="none" caption="Given"
  ></sl-diagram>
  <sl-diagram
    size="2x2" clues="1,1:3 2,2:3"
    emphasis="0,0->R; 0,0->D; 1,2->R; 2,1->D"
    continue="none" caption="Forced"
  ></sl-diagram>
</sl-sequence>

Fade — all sides

Default continue="all" fades on every side.

<sl-diagram
  size="2x2"
  clues="1,1:3 2,2:1"
  lines="0,0->RD"
></sl-diagram>

Fade — selective sides

continue="right bottom" fades only right and bottom.

<sl-diagram
  size="2x2"
  clues="1,1:3 2,2:1"
  lines="0,0->RD"
  continue="right bottom"
></sl-diagram>

CSS custom properties

Resized with --sl-cell-size: 72 and custom colors.

<div style="--sl-cell-size: 72;
     --sl-emphasis: #e03030;
     --sl-grid: #d8d8d8;
     --sl-clue: #333;">
  <sl-diagram
    size="2x2" clues="1,1:3 2,1:1"
    lines="0,0->RR"
    emphasis="0,0->D"
    continue="none"
  ></sl-diagram>
</div>
